# A2 Adults Curriculum

All 12 units of the A2 Adults track, each broken down lesson by lesson.
Every lesson is 25 minutes, using the adult-specific A2 lesson shape from
the level-jump/A2-approach document — practical and functional, not the
same shell as Kids or Teens: **Warm-up with real-life context (3 min) →
Short text task: a model text or functional example, given first (7 min)
→ Language noticing, drawn from that text (5 min) → Guided speaking or
writing / controlled practice (6 min) → Personalized task (2 min) →
Review/mini-project, with correction and reflection (2 min)**.

This is content/planning-stage only — none of it is built into the app
yet (no Supabase rows, no slide-type mapping, no images/audio).

**Correction made before saving** (same issue found in A2 Teens, fixed
the same way): the Perplexity draft used A2 Kids' deductive lesson order
(teach the language first, then give a text) instead of the
Adults-specific order this document requires (give the functional model
text first — an email, form, or dialogue — then notice the language
patterns in it). All 60 lessons were reordered — same activities and
materials, just the "Language in context" and "Short text task" rows
swapped, renamed to "Short text task (model text/functional example)"
and "Language noticing (from the text)" to reflect the adult-specific
terminology from the source document.

**Content check (no changes needed):** unlike A2 Teens, no duplicate
lessons were found — every unit's Lesson 3 is a simple-past narrative
lesson, but each is tied to a different real-life domain (health,
travel, work, culture, problem-solving), which is intentional spaced
practice of the same grammar target across contexts, not repetition.
Topics checked against A1 Adults (Greetings, Family, Home, Daily
Routine, Work, Shopping, Food, Technology, Travel, Health, Places in
Town, Appointments/Plans) — most A2 units are natural spiral revisits
(functional emails/forms/comparisons A1 didn't have), plus two genuinely
new units filling real gaps: Leisure/Community and People/Culture.
Grammar targets (simple past, comparatives, connectors, "going to"
future saved for the final units) match the A1→A2 jump, and functional
literacy content (emails, forms, appointment calls, complaints,
invitations, housing ads) is present throughout, matching the document's
adult-specific requirement closely. Timing verified at exactly 25
minutes on every lesson.

# Unit 1: Daily Communication and Personal Updates

## Lesson 1: Everyday Small Talk

| Part | Plan |
|------|------|
| Time | 25 minutes |
| Main aim | The student can manage simple everyday conversation and give short personal updates. |
| Materials | Conversation prompt slides, topic cards, sentence frames, mini check-in board |

| Stage | Time | Online material / activity | Teacher role |
|-------|------|----------------------------|--------------|
| Warm-up (real-life context) | 3 min | Short check-in with simple A1 review questions about name, day, and routine | Recycle known language and set a practical adult tone |
| Short text task (model text/functional example) | 7 min | Short dialogue reading between two adults meeting before work; student identifies greeting, update, and response | Guide reading for function and useful phrases |
| Language noticing (from the text) | 5 min | Teacher models short everyday exchanges about how the day is going, what is new, and simple follow-up questions | Present natural small-talk language adults can use immediately |
| Guided speaking or writing | 6 min | Speaking ladder with prompt cards for a 4-line small-talk exchange | Prompt fuller short turns and support natural follow-up questions |
| Personalized task | 2 min | Student gives one real personal update about today or this week | Encourage direct personal communication |
| Review / mini-project | 2 min | Mini check-in card with one useful phrase | Reinforce practical conversation language |
