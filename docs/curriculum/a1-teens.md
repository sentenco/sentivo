# A1 Teens Curriculum

All 12 units of the A1 Teens track, each broken down lesson by lesson.
Every lesson is 25 minutes, built for online 1:1 teaching, with the same
6-stage shape used across all Sentivo tracks: **Warm-up (3 min) →
Presentation (5 min) → Guided practice (7 min) → Freer practice (6 min) →
Review (2 min) → Wrap-up (2 min)**.

This is content/planning-stage only — none of it is built into the app
yet (no Supabase rows, no slide-type mapping, no images/audio).

Two things worth being aware of when this gets built:
- Units 4 (Friends and Free Time) and 5 (Hobbies and Interests) share
  some vocabulary overlap (reading, games) — inherited from the roadmap's
  own unit split, not an error.
- Unit 9 Lesson 2 ("is wearing") and Unit 12 Lesson 2 ("I'm going to...")
  use present continuous / going-to future, technically past strict A1
  grammar — but both are deliberately treated as fixed, unanalyzed
  phrases rather than taught grammar, a standard workaround for
  functionally necessary language.

# Unit 1: Greetings and Introductions

## Lesson 1: Hello and Me

| Part | Plan |
|------|------|
| Time | 25 minutes |
| Main aim | The student can introduce themself and exchange basic greeting information. |
| Materials | Slide deck, name card activity, text chat prompts, mini quiz slides |

| Stage | Time | Online material / activity | Teacher role |
|-------|------|----------------------------|--------------|
| Warm-up | 3 min | Quick hello routine plus on-screen name prompt | Build comfort, check mood, start speaking immediately |
| Presentation | 5 min | Simple greeting phrases, name, age, country, spelling model slides | Model language, highlight key phrase forms |
| Guided practice | 7 min | Drag-and-match: question and answer pairs for introductions | Correct answers, prompt repetition, support pronunciation |
| Freer practice | 6 min | Short live introduction exchange using personal details | Ask follow-up questions, encourage full simple answers |
| Review | 2 min | Quick recall slide with 3 target phrases | Confirm memory, recycle key language |
| Wrap-up | 2 min | Exit line: “Tell me one thing about you.” | End on success, give brief praise |

## Lesson 2: Questions and Answers

| Part | Plan |
|------|------|

| Main aim | The student can ask and answer basic introduction questions. |

| Stage | Time | Online material / activity | Teacher role |
|-------|------|----------------------------|--------------|
| Warm-up | 3 min | Review of greetings from Lesson 1 | Reactivate prior language, keep pace brisk |

| Wrap-up | 2 min | Short spoken recap: 2 questions + 2 answers | Reinforce confidence, close neatly |
