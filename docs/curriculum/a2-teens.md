# A2 Teens Curriculum

All 12 units of the A2 Teens track, each broken down lesson by lesson.
Every lesson is 25 minutes, using the teen-specific A2 lesson shape from
the level-jump/A2-approach document — genuinely different from A2 Kids,
not just a relabeled copy: **Warm-up (opinion/response) (3 min) → Short
text task: reading/listening/picture-text, given first (7 min) →
Language focus, drawn from that text (5 min) → Guided speaking or
writing (6 min) → Personalized task (2 min) → Review/mini-project (2
min)**.

This is content/planning-stage only — none of it is built into the app
yet (no Supabase rows, no slide-type mapping, no images/audio).

**Correction made before saving:** the Perplexity draft used A2 Kids'
lesson order (teach the language first, then give a text) instead of the
Teens-specific inductive order this document requires (give the text
first, draw the language focus out of it). All 60 lessons were reordered
— same activities, same materials, just the "Language in context" and
"Short text task" rows swapped and the language stage renamed "Language
focus (from the text)" to reflect that it now follows the input rather
than preceding it.

**Also fixed:** Unit 4 Lesson 2 was a near-duplicate of Unit 2 Lesson 3
(both were "describe last weekend using simple past," two units apart).
Replaced Unit 4 Lesson 2 with "How I Started My Hobby" — same simple-past
grammar target, but tied to the unit's actual hobby theme instead of
repeating a generic weekend recap. Unit 4's Lesson 5 review was updated
to match.

Unit topics were proposed by Perplexity, checked against the A1 Teens
topic list (Greetings, Family and People I Know, School Life, Friends and
Free Time, Hobbies and Interests, My Daily Routine, My Room, Food and
Drinks, Clothes and Style, Weather and Seasons, Social Media and Tech,
Weekend Plans). Several A2 topics revisit the same domains at a deeper
level (Friends, School, Hobbies, Food) — this is spiral progression, not
repetition: the A1 versions taught basic vocabulary and questions, the A2
versions add opinions, comparisons, past-tense storytelling, and short
written texts (posts, reviews, messages) that A1 didn't touch. Grammar
targets (simple past, comparatives, connectors, "going to" future saved
for the final unit) match the A1→A2 jump.

6. Food and Going Out

# Unit 1: Identity and Personal Style

## Lesson 1: My Style, My Choice

| Part | Plan |
|------|------|
| Time | 25 minutes |
| Main aim | The student can describe personal style and give simple reasons for clothing and style choices. |
| Materials | Style photo slides, opinion poll cards, sentence frame board, mini mood board |

| Stage | Time | Online material / activity | Teacher role |
|-------|------|----------------------------|--------------|
| Warm-up (opinion/response) | 3 min | Quick personal-choice poll with style images and emoji reactions | Activate interest and start with teen-relevant personal response |
| Short text task (reading/listening/picture-text) | 7 min | Short reading with two teen profile captions about clothes and style choices; student matches each profile to a picture set | Guide reading for main message and help the learner notice opinion language |
| Language focus (from the text) | 5 min | Photo collage introducing style words such as casual, sporty, neat, colorful, simple with model opinions | Present useful teen vocabulary in a relatable context and model because for reasons |
| Guided speaking or writing | 6 min | Speaking ladder: the student describes two styles and says which one fits them better | Prompt fuller answers and support sentence connection with and and because |
| Personalized task | 2 min | Student gives one true sentence about what they usually wear and why | Encourage a natural personal answer |
| Review / mini-project | 2 min | Mini digital mood board title plus one caption sentence | Wrap up with a visual identity task and recap target language |
